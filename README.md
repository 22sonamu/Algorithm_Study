### Intro

1. 최상위 디렉토리의 README.md 는 전체 안내 파일
2. week 디렉토리의 README.md 는 해당 주차에 문제를 풀면서 알게된 소소한 팁
3. 파일 이름은 `{사이트or책}_{레벨or난이도}_{문제번호}_{닉네임}.{파일형식}` 형식으로 작성 (ex. boj_G1_1010_Heegun.py)
